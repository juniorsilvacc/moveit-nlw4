# Como executar o projeto

## Pré-requitos: npm/yarn 

```bash
# clonar repositório  
git clone https://github.com/juniorsilvacc/moveit-nlw4.git

# entrar na pasta do projeto mobile
cd moveit-nlw4

# instalar dependências
yarn install

# executar projeto
yarn start
```
